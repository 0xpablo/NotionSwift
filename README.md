# NotionSwift

Unofficial Notion SDK for iOS & macOS 


## Supported Endpoints

### Databases
 * Retrieve a database ✅
 * Query a database ✅
 * List databases ✅
 
### Pages
* Retrieve a page ✅
* Create a page
* Update page properties

### Blocks 
* Retrieve block children
* Append block children

### Users
* Retrieve a user ✅
* List all users ✅

### Search 
* Search
